# Omnitech
Le projet Omnitech s’inscrit dans une démarche d’expérimentation autour de l’intelligence artificielle embarquée (Edge AI) et des interfaces homme-machine (IHM) de nouvelle génération. Inspiré de l’outil multifonction du jeu vidéo Mass Effect, ce dispositif vise à concevoir un système portable fixé sur l’avant-bras, capable d’analyser visuellement son environnement, d’interpréter les données captées et de communiquer oralement les résultats à l’utilisateur en temps réel.
